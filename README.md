# Bot-FInOps
Kali ini saya mendapat Project Kelompok Membuat Bot-FinOps untuk monitoring Resource Quota seperti CPU, dan Memory
saya menggunakan mengerjakan Project tersebut di lingkungan kubernetes, dengan alat seperti, kubectl, minikube, Helm, Prometheus, Grafana, Workflow N8N, alertmanager, dan Slack.
Tujuan Project Project ini adalah unutk memenuhi indikasi tugas akhir dari Program studi NFA.
Project ini memiliki bebrapa fungsi yang pasti yaitu:
1. Untuk manajemen Resource yang optimal
2. menjadi contoh implementasi yang bisa digunakan untuk pmebelajaran di NFA
3. untuk menguji kompetensi peserta program studi independen NFA.
